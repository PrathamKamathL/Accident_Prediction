Accident Severity Prediction

1.	Input and Output Specification:  
    a. Input: Details of the accident, with a total of 18 features.  
    b. Output: Multiclass classification of severity into three classes:  
    *	Fatal injury – class 0  
    *	Serious injury – class 1    
    *	Slight injury – class 2   
2.	Source of the Dataset: https://www.kaggle.com/datasets/saurabhshahane/road-traffic-accidents , referenced by: https://data.mendeley.com/datasets/xytv86278f/1 
3. Steps to run application:  
    a. Fork the repository using 'Fork'  
    b. Clone the repository
